BlackDuck Integration — 3 Approaches

Where and how Black Duck Detect + Hub fit into this lab

A

Recommended — Detect-only Docker container (no Hub)

Run blackducksoftware/detect as a one-shot Docker container in the Jenkins stage. Detect scans the built image layers and dependency files, writes a JSON/text report to the workspace, and exits. No persistent Hub server needed.

Pros

  • Simple — single container, no infra overhead
  • Fast to set up
  • No license server required
  • Follows lab's Docker Compose pattern

Cons

  • No persistent UI to browse findings over time
  • Results live only in Jenkins console + artifact files
B

Full Black Duck Hub via Docker Compose

Stand up the complete Black Duck Hub stack (hub-webapp, hub-postgres, hub-registration, hub-scan, hub-jobrunner) via Docker Compose. Detect connects to it. Results visible in Hub's full web UI.

Pros

  • Full production-like UI and API
  • Policy management, license compliance, project history
  • Most complete learning experience

Cons

  • Requires a valid Black Duck license/activation key
  • Heavy: 6+ containers, 8-16 GB RAM
  • Port management complexity (adds 8083+)
  • Slow startup (5-10 min)
C

Detect + lightweight SBOM viewer (Syft + Grype)

Use Syft to generate an SBOM from the Docker image, Grype to scan it for CVEs, and serve results via a simple HTML report. Conceptually equivalent to Black Duck Detect but fully open-source and zero-license.

Pros

  • Zero licensing friction
  • Very fast and lightweight
  • Generates industry-standard SBOM (SPDX/CycloneDX)

Cons

  • Not actually Black Duck — different tool, different learnings
  • No Black Duck-specific Jenkins plugin experience

My recommendation

Option A fits the lab best: it adds Black Duck Detect as a real Jenkins stage using the official Docker image, follows the existing Docker-based pattern, and requires no license server. Option B is the most complete but the licensing and resource cost is high for a learning lab. Option C is a good fallback if Black Duck Detect requires a license key at runtime.
